# BaiduFace
纪念我第一个成功上线的作品（人脸识别签到系统 系统后端+后台代码  
由于是第一次尝试开发实际上线的产品，很多方面的考虑都不够全面  
再加上后来种种原因，开发出来的产品不能投入使用，技术、市场方面都有设计到  
但是这个项目对于我来说，意义还是挺大的  
由于小程序的开发者不是本人，所以就没有把相关代码上传到github了  
感谢跟我合作的那位小伙伴以及项目负责人给我这次机会
